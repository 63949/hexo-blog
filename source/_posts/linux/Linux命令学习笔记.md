## 查看磁盘占用

```
df: 查看硬盘使用情况
  -h 用方便阅读的方式显示
  -T 显示文件系统类型
  -l 只显示本地文件系统

du: 查看目录或文件所占空间
  -h 用方便阅读的方式显示
  -a 显示所有文件夹与文件的大小（默认不显示文件的大小）
  -s 只显示总和
  -d N 在显示文件夹/文件时，只显示深度<=N
  --time 显示最后修改时间
  --exclude=PATTERN 排除符合PATTERN的路径
  -b 显示为字节数。
  -k 显示为K字节数。
  -m 显示为M字节数。
```
